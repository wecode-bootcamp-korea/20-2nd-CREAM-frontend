# 🍦Team CREAM

## ✅ 프로젝트 소개

- 프로젝트 개요: CREAM(Code Rules Everything Around me) 팀의 KREAM 클론 코딩

- 프로젝트 기간: 2021.05.24 ~ 2021.06.04

<br>

## 🎯 기획 목표

- 한정판 중고 **의류품 거래**를 위한 **중개 플랫폼**

- 가격별 제품 **오름**차순, **내림**차순

- 브랜드, 사이즈, 색상 및 컬렉션별 제품 **필터링**

- 단순 구매 및 판매 + **입찰 시스템** 구현

- 입찰 및 체결 거래 **데이터 실시간 그래프화**

<br>
